# 3️⃣ CRUD модул за clubs (`clubs_service.py`)

## Да има функции:

* `add_club(name)`
* `get_all_clubs()`
* `delete_club(name или id)`
* `update_club(...)` *(по желание, но препоръчително)*

## Валидация:

* име ≠ празно
* без дублирани клубове

---

# 4️⃣ Chatbot (първа версия)

## Задължителна логика:

```text
Chat loop:
input → intent → handler → response
```

## `intents.json` (минимум intents)

* `help`
* `exit`
* `add_club`
* `list_clubs`
* `delete_club`

## Примерни команди:

* Добави клуб Левски София
* Покажи всички клубове
* Изтрий клуб Ботев
* помощ
* изход

## Разпознаване:

* чрез regex правила
* извличане на параметри
