# ✅ Етап 8: AI Модул (Прогноза на мач)

---

# 1️⃣ 🎯 Цел на етапа

Да се реализира AI модул, който:

- анализира исторически резултати
- изчислява текуща форма на отборите
- прогнозира вероятности за:
  - 🏠 Победа домакин
  - 🤝 Равен
  - 🛫 Победа гост
- работи чрез чатбот команда

---

# 2️⃣ Минимално изискване (задължително)

Системата трябва да може да изпълни команда:

```text id="d8n3qy"
Прогноза Левски срещу Лудогорец
````

---

## Примерен резултат

```text id="v5k1zr"
Победа Левски: 32%
Равен: 27%
Победа Лудогорец: 41%
```

---

# 3️⃣ Подходи за реализация (ученикът избира 1)

---

# 🟢 Вариант A — Rule-Based модел (задължителен минимум)

## Без машинно обучение

Моделът трябва да използва:

* Последни `N` мача *(минимум 5)*
* Средни голове `(GF / GA)`
* Текущо класиране *(позиция)*
* Домакинско предимство *(bonus)*

---

## Примерна логика

```text id="w7m4xt"
Форма =
(точки от последните 5 мача / 15)
```

```text id="u2q8lh"
Атакуваща сила =
средно вкарани голове
```

```text id="b6r1vc"
Защитна сила =
средно допуснати голове
```

```text id="k9f5pa"
Индекс =
комбинация от горните показатели
```

---

## След това

* нормализиране до `100%`
* закръгляване

---

# 🔵 Вариант B — ML модел (за „Отличен“)

## Използване на `scikit-learn`

### Минимум

* `LogisticRegression`

или

* `RandomForestClassifier`

---

## Feature-и (минимум 3)

* разлика в точки
* разлика в голова разлика
* средно голове домакин
* средно голове гост
* форма последни 5 мача

---

## Target

| Стойност | Значение |
| -------- | -------- |
| `0`      | домакин  |
| `1`      | равен    |
| `2`      | гост     |

---

* `matches (played)`
* `league_teams`
* `standings`

  * *(или изчисляване в реално време)*
* `goals`

  * *(по избор)*

# 6️⃣ Chatbot интеграция

## Команда

```text id="t5p8zn"
Прогноза <Отбор1> срещу <Отбор2>
```

---

## Handler

Трябва да:

* провери дали отборите са в една и съща лига
* извлече feature-и
* извика `ai_service`
* върне резултат

---

# 7️⃣ Acceptance Criteria (проверими условия)

* ✔ Работи при минимум `5` изиграни мача
* ✔ Работи само за съществуващи отбори
* ✔ Връща `3` вероятности
* ✔ Сборът им ≈ `100%`
* ✔ Няма отрицателни проценти

---

# 8️⃣ Минимални тестови сценарии

| Сценарий                      | Очакван резултат |
| ----------------------------- | ---------------- |
| Прогноза при достатъчно данни | OK               |
| Прогноза при `< 5` мача       | ERROR            |
| Несъществуващ отбор           | ERROR            |
| Отбори от различни лиги       | ERROR            |
| Проверка: сбор = 100%         | OK               |

---

# 9️⃣ Критерии за оценяване

| Критерий                 | Тежест |
| ------------------------ | ------ |

| Чатбот интеграция        | 20%    |
| Кодова структура         | 15%    |
| Документация             | 10%    |
