# ЕТАП 3 — CRUD „Футболисти“ + Филтри

## Цел

Да се реализира пълен модул `players` с връзка към таблицата `clubs`.

Системата трябва да позволява:
- управление на футболисти
- връзка между играчи и клубове
- филтриране по клуб
- chatbot команди за работа с играчи

---

# Основни изисквания

## Таблица `players`

Таблицата трябва да съдържа:

| Поле | Описание |
|---|---|
| id | PRIMARY KEY |
| club_id | FOREIGN KEY към clubs |
| full_name | Име на играча |
| birth_date | Дата на раждане |
| nationality | Националност |
| position | Позиция |
| number | Номер |
| status | Статус |
